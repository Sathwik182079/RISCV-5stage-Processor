# 5-Stage Pipelined RISC-V Processor

## Project Description

This project presents the design and implementation of a **32-bit
5-stage pipelined RISC-V processor** in **Verilog HDL**

The processor implements the classic **Instruction Fetch (IF),
Instruction Decode (ID), Execute (EX), Memory Access (MEM), and Write
Back (WB)** pipeline, enabling multiple instructions to execute
simultaneously for improved throughput. It includes the fundamental
components required for a pipelined processor such as the Program
Counter, Register File, ALU, Control Unit, Immediate Generator,
Instruction Memory, Data Memory, and dedicated pipeline registers
between each stage.

To ensure correct execution of dependent instructions, the processor
incorporates a **Hazard Detection Unit** and a **Forwarding Unit**,
allowing it to resolve data hazards through stalling and operand
forwarding. Control hazards generated by branch and jump instructions
are handled using pipeline flushing, supporting correct execution of
**Branch**, **JAL**, and **JALR** instructions.

The design has been functionally verified using **Xilinx Vivado** with
custom Verilog testbenches covering arithmetic, logical, immediate,
branch, jump, forwarding, hazard, and register write-back operations.
The project demonstrates a practical implementation of pipelined
processor design concepts widely used in modern RISC processors.

## Key Features

-   32-bit RISC-V (RV32I educational subset) processor
-   Five-stage pipelined architecture (IF, ID, EX, MEM, WB)
-   Program Counter and Next-PC logic
-   Register File (32 × 32-bit)
-   Immediate Generator
-   Arithmetic Logic Unit (ALU)
-   ALU Control Unit
-   Main Control Unit
-   Instruction Memory
-   Data Memory
-   Pipeline Registers (IF/ID, ID/EX, EX/MEM, MEM/WB)
-   Hazard Detection Unit
-   Data Forwarding Unit
-   Branch and Jump (JAL/JALR) support
-   Pipeline stalling and flushing
-   Functional simulation and synthesis using Xilinx Vivado

## Verification

-   Logical operations
-   Shift operations
-   Immediate instructions
-   Register write-back
-   x0 register protection
-   Data forwarding
-   Load-use hazard handling
-   Branch execution
-   Jump instructions (JAL/JALR)
